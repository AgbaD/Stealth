# Stealth
A script to hide or unhide files or folders. The complete path to the file must be provided

## Usage
   - 1. ```
        $ python3 stealth.py
        ```
   - 2. Enter file path
   - 3. Enter command (hide or unhide)
   - 4. To unhide, enter path to folder + file name without '.'
        ```
        $ /home/blank/Desktop/{file name}
        ```
